# AJMI1 Support & Documentation

This repository hosts the official support and documentation resources for the **AJMI1 Calculator**, a clinical decision-support tool for evaluating the severity of cervical myelopathy using the AJMI scoring system (Ambulation, Jerks, Motor, Incontinence, Imaging).

## About AJMI1
The AJMI1 Calculator assists healthcare professionals by providing a structured scoring method to support clinical decision-making. The app functions entirely offline and does not collect, store, or transmit any personal or patient data.
